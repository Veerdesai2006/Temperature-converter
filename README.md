# Temperature-converter
## 📊 Overview This project is a simple Python-based temperature converter that converts temperature values between Celsius, Fahrenheit, and Kelvin.  This project shows how basic programming concepts can be utilized to solve real-world problems.
## 🚀 Features
- Convert Celsius → Fahrenheit & Kelvin  
- Convert Fahrenheit → Celsius & Kelvin  
- Convert Kelvin → Celsius & Fahrenheit  
- User-friendly input and output  



## 🧠 Concepts Used
- Functions  
- Conditional statements (if-else)
- Mathematical operations

 ## 💻 Tech Stack
- Python
